# Atol Test Task

Тестовое задание.
Настроен action для github: проверка на psr12 и запуст автотеста.

## Requirements

- PHP >= 8.0.22
- Composer >= 2.2.18

## Install

```
composer install
```

## Run

```
php run.php
```

## Test

```
composer run-script ci:php:test
```

## Shiff

```
composer run-script ci:php:sniff
```
